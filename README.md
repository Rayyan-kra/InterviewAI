# InterviewAI 🚀

An AI-powered interview preparation platform built using the MERN Stack and Gemini AI.

## 📌 Overview

InterviewAI helps candidates prepare for technical interviews by analyzing resumes, generating interview questions, identifying skill gaps, and creating personalized preparation roadmaps.

## ✨ Features

* Resume Analysis
* AI-Generated Interview Questions
* Detailed Answer Explanations
* Skill Gap Detection
* Interview Readiness Score
* Personalized Learning Roadmap
* Secure Authentication (JWT)
* Modern Responsive UI

## 🛠 Tech Stack

### Frontend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rayyan-kra/InterviewAI.git
cd InterviewAI
```

### Backend Setup

```bash
cd Backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

## 🔐 Environment Variables

Create a `.env` file inside the Backend folder.

```env
MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
GOOGLE_GENAI_API_KEY=your_api_key
```
